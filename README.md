# ClarityPath AI

Hackathon MVP for **AI for Bharat**.

Phone-first **PWA** for Tier-2/3 India students to escape “tutorial hell” by completing small, verifiable tasks.

## MVP (What it does)
- **Single learning path:** Frontend Web Basics
- **Daily tasks:** Parsons problems with **deterministic validation** (no AI in daily loop)
- **Offline:** read-only (tasks viewable offline)
- **Online:** submit + verify (sync required)
- **No videos. No certificates.**
- **Public transcript:** evidence of completed + verified tasks (timestamps)

## Demo (5 minutes)
1) Open PWA (cached)
2) Complete 1 Parsons task (offline)
3) Go online → Sync → Verified
4) Open transcript link `/t/{userId}`
